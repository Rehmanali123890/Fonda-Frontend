# 🎨 Fonda Frontend

🔗 Website: https://www.mifonda.io/

Fonda Frontend is a modern **Angular-based web application** that provides a unified dashboard for restaurants to manage orders, payments, and operations across multiple delivery platforms.

👉 Built for simplicity, speed, and real-time control.

---

## 📌 Overview

The Fonda frontend connects with the backend APIs to provide merchants with a **single interface** to:

- Manage orders from multiple delivery platforms  
- Track payments and transactions  
- View analytics and insights  
- Handle day-to-day restaurant operations  

💡 No need to switch between different apps — everything is centralized.

## 🔄 Workflow

1. Frontend fetches data from backend APIs  
2. Displays orders from all platforms  
3. Merchant interacts via dashboard  
4. Actions (accept/reject orders) sent to backend  
5. Real-time updates reflected in UI
